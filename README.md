OWD Book Reader
===============

This is a HTML5 ebook reader, currently supporting ePub format, for the OWD (Open Web Device, aka HTML5 phone)


Original idea
-------------

Based on the HTML5 paginator [Monocle](http://monocle.inventivelabs.com.au/), we are creating a html5 application
that is able to manage, read and show epubs files using Monocle as visualisation.
